■これは何をするものか

マウスカーソルの位置の変更をするSAORIです。

■動作環境

・Win95/NT4以上

■使用方法

・呼び出し
　Argument0　X座標
　Argument1　Y座標

・結果
　Result	通常はありません。エラー時のみエラーコードが返ります。
